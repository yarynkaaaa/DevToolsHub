Категорії ресурсів

  • Редактори та середовища розробки

    Популярні IDE та текстові редактори: VS Code, JetBrains, Sublime Text.

  • Навчальні курси та документація

    Онлайн-курси, корисні книги, технічні гайди, ресурси для практичного застосування.

  • Плагіни та розширення

    Практичні інструменти для покращення якості коду та спрощення розробки.

  • Інструменти продуктивності

    Сервіси для керування кодом, спільної роботи та організації процесів.

Ресурси для твоєї професії

  • Web designer

    Для створення візуально привабливих та зручних для користувача веб-сайтів: Figma, Adobe XD, Dribbble, Behance.

  • Front-end developer

    Розробка інтерфейсу кормстувача: HTML, CSS, JavaScript, MDN Web Docs, CSS Tricks, FreeCodeCamp, Frontend Mentor .

  • Back-end developer

    Робота з серверною частиною сайтів, керування базами даних та логікою програми: Node.js Docs, PHP.net, PostgreSQL, Laravel .

  • Fullstack developer

    Поєднання як фронтенду, так і бекенду, для створення повноцінних веб-додатків: The Odin Project, Fullstack Open, Roadmap.sh.

  • QA-tester

    Тестування програмного забезпечення, перевірка його якості та функціональності: ISTQB, Test Automation University, Guru99.

  • Cyber specialist

    Захист системи від атак хакерів, аналіз загроз та впровадження заходів безпеки: Hack The Box, OWASP, Cybrary, TryHackMe.
